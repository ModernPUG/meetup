# php-tetris

PHP로 만든 테트리스

### 프로젝트의 목적
- PHP의 다양한 활용
- 게임 프로그래밍의 기초 학습
- 테트리스의 기본 로직 이해

### 실행 환경
- PHP 5.6.x
- Mac OS X 터미널
- 타일 출력은 [유니코드 이모지](http://unicode.org/emoji/charts/full-emoji-list.html) 기반으로 운영체제와 터미널 환경에 따라 출력되는 모습이 달라진다.
